floccus

Sync your bookmarks across browsers via a self-hosted server

floccus is a browser extension for Firefox and Chrome-based browsers which allows you to synchronize your bookmarks across browsers and devices via a self-hosted server of your choosing. Full control, full privacy, full access to your favorite experiences on the web.

Sync backends

  • Nextcloud Bookmarks
    Nextcloud is a self-hosted sync and share solution which also offers a bookmarks app, which allows you to access your bookmarks via a beatiful web UI and to sync them with your browser via floccus.
  • WebDAV
    If you have a WebDAV file server at hand, like any version of nextcloud/owncloud, box.com or with any other WebDAV server, commercially or self-hosted, you can sync your bookmarks using floccus.
  • Local file and more
    You can also simply sync with a local file, using a companion desktop app, allowing you to sync that file to other computers using your favorite file syncing solution, like Dropbox, Syncthing, rsync, etc.
